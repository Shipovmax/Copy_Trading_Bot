### 1. Clone the repository or download the files
```bash
git clone https://github.com/Shipovmax/Copy_Trading_Bot.git
cd Copy_Trading_Bot
```

### 2. Install dependencies
```bash
# macOS/Linux
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# Windows
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

### 3. Run the bot
```bash
python bot.py
```
