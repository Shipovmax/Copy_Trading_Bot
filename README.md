### 1️⃣ Клонируй репо или скачай файлы:
```bash
git clone https://github.com/Shipovmax/Copy_Trading_Bot.git
cd Copy_Trading_Bot
```

### 2️⃣ Установи зависимости:
```bash
# macOS/Linux
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# Windows
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

### 3️⃣ Запусти бота:
```bash
python bot.py
```
