# Copy Trading Bot PRO

Telegram бот для анализа криптовалют с использованием технического анализа и AI-помощника Perplexity.

## 📋 Описание

**Copy Trading Bot** — это полнофункциональный Telegram бот для трейдеров криптовалют, который предоставляет:

- **📊 Технический анализ** криптопар с расчетом RSI, MACD и Moving Averages
- **🤖 AI-анализ** через Perplexity для получения интеллектуальных рекомендаций
- **💰 Управление балансом** на бирже OKX (спот/маржин/фьючерс)
- **📢 Автоматические сигналы** с уведомлениями о BUY/SELL сигналах
- **🔑 Безопасное хранение** API-ключей локально (config.json)
- **⏰ Гибкий таймфрейм** анализа (1m, 5m, 15m, 1h, 4h, 1d)

Все данные пользователя хранятся локально на сервере, не передаются третьим лицам.

## ⚙️ Конфигурация

### Получение API ключей

**Telegram:**
1. Напиши [@BotFather](https://t.me/botfather)
2. Команда `/newbot` для создания нового бота
3. Скопируй токен из ответа

**OKX:**
1. Зайди на [okx.com](https://www.okx.com)
2. Account → API Management → Create API Key
3. Выбери разрешения (для sandbox режима достаточно минимальных)
4. Скопируй API Key, Secret Key, Passphrase

**Perplexity (опционально):**
1. Зайди на [perplexity.ai/api](https://www.perplexity.ai/api/)
2. Создай API ключ
3. Выбери нужный план

## 🎯 Основные функции

### 📊 Анализ криптопар

```
/start → 📊 Анализ → Выбери пары → 🔍 АНАЛИЗИРОВАТЬ
```

- Выбор до 20 топовых крипто пар (BTC, ETH, BNB и т.д.)
- Расчет технических индикаторов:
  - **RSI** (Relative Strength Index)
  - **MACD** (Moving Average Convergence Divergence)
  - **Moving Averages** (MA20, MA50, MA200)
- AI-анализ через Perplexity с рекомендациями

### 💰 Управление балансом

```
/start → 💰 Баланс → Выбор счета
```

- Просмотр баланса USDT
- Список активов (топ-20)
- Поддержка типов счетов:
  - Спот (Spot Trading)
  - Маржин (Margin)
  - Фьючерс (Futures)
- Автоматическое обновление баланса

### 📢 Автоматические сигналы

```
/start → 📢 Сигналы → ✅ Включить/Выключить
```

- Настройка интервала проверки (5, 15, 30, 60 минут)
- Выбор пар для мониторинга
- Получение уведомлений:
  - 🟢 **BUY** — сигнал покупки
  - 🔴 **SELL** — сигнал продажи
  - ⚪ **HOLD** — ждать

### ⏰ Таймфрейм

```
/start → ⏰ TF → Выбери таймфрейм
```

Поддерживаемые таймфреймы:
- 1m, 5m, 15m (краткосрочная торговля)
- 1h, 4h (среднесрочная торговля)
- 1d (долгосрочная торговля)

### 🤖 AI-помощник

```
/start → 🤖 AI → Выбери вопрос
```

Предустановленные вопросы:
- 💡 **Покупать?** — Топ-5 монет для покупки
- 📊 **Индики** — Объяснение RSI, MACD, MA
- 🎯 **Стратегия** — Лучшие стратегии для новичков
- 📈 **Риск** — Управление риском в трейдинге

## 📊 Примеры использования

### Анализ BTC на 1 час

1. Нажми `📊 Анализ`
2. Выбери `BTC/USDT` (нажми на ней)
3. Убедись, что таймфрейм `1h` (⏰ TF)
4. Нажми `🔍 АНАЛИЗИРОВАТЬ`
5. Получи результат с техническим анализом и AI-рекомендацией

### Включение автосигналов

1. Нажми `📢 Сигналы`
2. Нажми `✅ Включить/Выключить` (вкл.)
3. Установи интервал `⏱️ Интервал` (например, 5 минут)
4. Выбери пары `📋 Пары` (BTC, ETH и т.д.)
5. Бот будет отправлять сигналы автоматически

### Добавление OKX ключей

1. Нажми `🔑 API`
2. Нажми `🔐 OKX API Key` и добавь ключ
3. Нажми `🔑 OKX Secret` и добавь secret
4. Нажми `🎯 OKX Passphrase` и добавь passphrase
5. Нажми `✅ Проверить OKX` для проверки

## 🔐 Безопасность

- **Локальное хранилище:** Все API ключи сохраняются только локально в `user_configs.json`
- **Шифрование:** Рекомендуется использовать переменные окружения для критичных ключей
- **Sandbox режим:** По умолчанию бот работает в sandbox режиме OKX для безопасности

## 🐛 Решение проблем

### Ошибка: "Бот не отвечает"

```bash
# Проверь интернет
ping api.telegram.org

# Проверь токен (в BotFather: /token)
# Убедись что скопировал ровно токен без пробелов
```

### Ошибка: "API Key invalid"

1. Проверь в OKX что ключ еще активен (Account → API Management)
2. Проверь что скопировал ровно без пробелов
3. Убедись что выбрал правильные разрешения

### Ошибка: "Перplexity API недоступен"

1. Проверь что ключ Perplexity добавлен (🔑 API → 🤖 Perplexity)
2. Проверь баланс аккаунта на https://www.perplexity.ai/api/
3. Убедись что есть интернет-соединение

### Анализ зависает

Если анализ долго загружается (>30 сек):
- Биржа может быть перегружена
- Проверь интернет-соединение
- Попробуй заново через 5 минут
