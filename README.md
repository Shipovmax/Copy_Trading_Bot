## 📥 УСТАНОВКА PERPLEXITY KEY

В коде строка:
```python
headers={"Authorization": "Bearer ppl_..."}
```

Нужно вставить свой ключ от Perplexity:

1. Зайди на https://www.perplexity.ai/api/
2. Создай API ключ
3. Замени `ppl_...` на свой ключ
4. Всё! Теперь AI будет работать

**Без ключа**: AI будет просто говорить что не может подключиться
