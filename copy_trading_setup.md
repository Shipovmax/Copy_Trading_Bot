# 🤖 Copy Trading Bot для OKX - Полная Инструкция

## Структура проекта
```
copy-trading-bot/
├── main.py                    # Главный файл для запуска
├── config.py                  # Конфиг (API ключи, параметры)
├── trader_scanner.py          # Сканер топ-трейдеров OKX
├── position_monitor.py        # Мониторинг позиций
├── copy_executor.py           # Исполнение копи-ордеров
├── risk_manager.py            # Управление рисками (стоп-лоссы)
├── telegram_notifier.py       # Уведомления в Telegram
├── database.py                # Логирование сделок в SQLite
└── requirements.txt           # Зависимости Python
```

## Установка

### 1. Клонируй или создай папку:
```bash
mkdir copy-trading-bot
cd copy-trading-bot
```

### 2. Создай виртуальное окружение:
```bash
python3 -m venv venv
source venv/bin/activate  # на macOS/Linux
# или
venv\Scripts\activate  # на Windows
```

### 3. Установи зависимости:
```bash
pip install -r requirements.txt
```

### 4. Создай .env файл с ключами OKX:
```bash
cp .env.example .env
# Отредактируй .env с твоими OKX API ключами
```

### 5. Запусти бота:
```bash
python main.py
```

## Как получить OKX API ключи?

1. Зайди на OKX.com → Account → API
2. Создай новый ключ (Read-Only для безопасности)
3. Копируй: API Key, Secret Key, Passphrase
4. Положи в `.env` файл

## Как работает бот?

1. **Сканирует топ-трейдеров** на OKX Leaderboard (ROI > 100%)
2. **Мониторит их позиции** каждые 30 сек
3. **Копирует новые ордеры** с коэффициентом 0.1x (меньше риск)
4. **Ставит стоп-лоссы** на 3-5% от входа
5. **Отправляет уведомления** в Telegram
6. **Логирует все сделки** в базу данных

## Риски и Особенности

⚠️ **ВАЖНО:**
- Бот копирует позиции, не гарантирует профит
- Топ-трейдеры иногда ошибаются (их ROI в прошлом не гарантирует будущее)
- Минимальный депозит на OKX: $100 USDT
- Газ и комиссии: ~0.1% за сделку
- Рекомендуемый старт: $500–1000 USDT

## Стратегия использования

### Неделя 1: Тестирование
- Запусти с коэффициентом 0.05x (половина от рекомендуемого)
- Мониторь 5-7 дней
- Смотри, ловит ли бот хорошие трейды

### Неделя 2-4: Масштабирование
- Если PNL позитивный (>+2%) → увеличь коэффициент до 0.1x
- Добавь депозита если нужно
- Рестартуй бота каждый понедельник

### Месяц 2+: Оптимизация
- Выбери только 2-3 лучших трейдеров (не все подряд)
- Анализируй какие пары копировать лучше (BTC/ETH/SOL)
- Может быть, автоматическая фильтрация по ROI >150%

## Документация кода

Каждый файл имеет подробные комментарии. Основные функции:

- `scan_top_traders()` - поиск лучших трейдеров
- `monitor_positions()` - мониторинг в реальном времени
- `execute_copy_order()` - исполнение ордера
- `set_stop_loss()` - установка стоп-лоссов
- `send_telegram_alert()` - уведомления

## Поддержка и Обновления

2. Убедись что интернет стабилен
3. Посмотри логи в `logs/` папке
4. Запусти с `--debug` флагом для подробного логирования
